# QuitClock

> 지금 이 순간 얼마 벌었어?

출근 시간이랑 월급 입력하면 실시간으로 얼마 벌었는지 보여주는 사이드 프로젝트.
정년까지 몇 초 남았는지도 같이.

## 기능

- 실시간 급여 카운터 (초당 원 단위)
- 직장생활 진행률 바
- 정년까지 남은 시간 (년/월/일/시간/분/초)
- 입력 정보는 서버에 저장되지 않음 (브라우저에서만 동작)

## 로컬 실행

그냥 `index.html` 브라우저에서 열면 됨.

```bash
# 또는 로컬 서버로 실행
npx serve .
```
